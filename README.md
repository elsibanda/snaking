# 🐍 Snake Game (JavaScript)

This is a simple Snake Game built using JavaScript, HTML, and CSS. The purpose of this project was to strengthen my understanding of core programming fundamentals and browser-based game development.

## What I learned
- JavaScript fundamentals (variables, functions, arrays, and logic)
- DOM manipulation and event handling
- Game loops and real-time updates
- Collision detection and score tracking
- Structuring small projects with clean, readable code

## Project goal
The goal was to practice problem-solving and improve my ability to build interactive applications using vanilla JavaScript without external libraries.

## Features
- Player-controlled snake movement
- Food generation and score tracking
- Game over detection
- Simple and responsive design

This project demonstrates my foundational JavaScript skills and my ability to build interactive front-end applications.
